## How use it

### Getting pass from URL to internal memory
```dart
import 'package:pass_flutter/pass_flutter.dart';

Pkpass_Downloader pkpass_Downloader = await Pass().saveFromUrl(url: 'https://link_to_pass/pass.pkpass');
```
